Splash Screen

Technical Description

The splash screen is built using HTML5, CSS3, and JavaScript, featuring a full-screen responsive layout with Flexbox, smooth fade-in animations, and a bubbling effect for an immersive underwater theme. A glassmorphism-style mission card enhances aesthetics, while JavaScript handles dynamic bubble generation and automatic redirection after 4 seconds.

Accessibility

Accessibility was ensured with semantic HTML, proper alt attributes, high contrast, and smooth, controlled animations for readability and user comfort.
